Welcome to OrdWatch

OrdWatch is a simple Telegram notification service that tracks Bitcoin addresses for new transactions in the mempool. It's designed to connect to the public Mempool API or your own Bitcoin node (coming soon) to send notifications when new transactions appear or get confirmed.

Features

  • Track multiple Bitcoin addresses.
  • Get notifications for unconfirmed and confirmed transactions via Telegram.
  • Connect to your own Bitcoin node or use the public Mempool API.
  • Customizable cron job frequency for transaction checks.
  • Posts links to incoming transactions via Mempool.space and detailed information about ordinals, runes, or BRC-20 via Ordiscan.io.

How to Run OrdWatch

This first version of OrdWatch runs inside a Docker container, which you have to run on your own machine or server. This version is self-hosted and requires you to have Docker installed. Once you have Docker set up, you can pull the OrdWatch repository, build the container and run it to track your Bitcoin addresses for new transactions.

For detailed instructions please checkout GitHub.
